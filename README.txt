* Comando do git *

git status -> Saber o status do git
git add (nome do arquivo) -> serve para adicionar o arquivo na pasta git, quando o arquivo for novo
git commit -a -m (mensagem de commit do que fez)
git push -> Para subir o git para o gitHub

* clonando arquivos *

 git clone https://github.com/EmanuelHSC/studying_git-github_1.git


* Renomeando arquivos *

git mv (de onde esta) (renomear o mesmo arquivo que esta no mesmo lugar)


* movendo arquivos *

git mv (de onde) (para onde)

* brench *

git branch (nome) -> ve as branch

git checkout (nome) -> vai para a branch

git checkout -b (nome) -> cira e vai pra branch
